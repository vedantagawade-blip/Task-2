# AI & ML Task 2: Feature Engineering, Model Optimization and Performance Comparison

## Overview
This project focuses on improving Machine Learning model performance through feature scaling, preprocessing, model training, and performance comparison.

## Dataset
- California Housing Dataset
- Source: Scikit-Learn

## Models Used
- Linear Regression
- Ridge Regression
- Decision Tree Regressor

## Evaluation Metrics
- RMSE (Root Mean Squared Error)
- R² Score

## Workflow
1. Import Libraries
2. Load Dataset
3. Data Exploration
4. Feature Scaling using StandardScaler
5. Train-Test Split
6. Train Multiple Models
7. Compare Performance
8. Select Best Model
9. Visualize Results
10. Draw Conclusions

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Google Colab

## Outcome
The project compares multiple regression algorithms and identifies the best-performing model for house price prediction based on RMSE and R² Score.
